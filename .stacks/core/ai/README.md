# Stacks AI

This package contains a pretty wrapper for the OpenAI API.

## ☘️ Features

- 🔎 Pretty API wrapper
- 🤷🏼‍♂️ git Diff Interpreter
- 🪐 GitHub integration
- 💨 Zero-config
- 🐶 Buddy CLI commands

## 🤖 Usage

```bash
pnpm i -D @stacksjs/ai
```

You may now use:

```ts
import { useAI } from '@stacksjs/ai'

const client = useAI()
// client...
```

Learn more in the docs.
